# Floating Hero Section

A modern floating hero section built with **Next.js**, **Tailwind CSS**, and **Framer Motion**, inspired by a premium legal-tech SaaS landing page.

## ✨ Features

- Responsive split-layout hero section
- Reusable floating card components
- Smooth floating animations with Framer Motion
- Dark mode support
- Modern SaaS-inspired UI
- Clean component architecture
- Tailwind CSS styling
- Mobile responsive design

---

## 🛠 Tech Stack

- Next.js 13+
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Ronnit44/Next.js-task
```

### 2. Navigate into the project

```bash
cd project
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start development server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---

## 🌙 Dark Mode

Dark mode is implemented using `next-themes`.

Use the theme toggle button in the top-right corner to switch themes.
